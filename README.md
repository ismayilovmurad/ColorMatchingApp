# Color Match
Your goal is to place the sliders as close as possible to the target color. The closer you are, the more points you score.

Frameworks:
- Foundation
- SwiftUI

Classes/Structures:
- State
- Binding
- ZStack
- VStack
- HStack
- Group
- Capsule
- Text
- Circle
- Color
- View
- Button
- Alert
- Slider
- ButtonStyle
- GeometryReader
